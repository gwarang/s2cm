# s2cm
S2CM (Simple docker Stack Container Manager)

## 소개
Docker swarm mode 로 구성한 클러스터의 manager 노드와 같은 서버에서 구동되는 Django 로 작동하는 웹을 통해 Docker Container를 배포 및 관리해 볼 수 있는 시스템
